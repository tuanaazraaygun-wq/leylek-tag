**Key Technical Concepts**
-   **Frontend-based Logic:** Shifting business logic (like location checks) to the client to reduce server load and improve perceived performance.
-   **Hybrid QR System:** Architectural pattern combining a fast, static QR code for user identification with a secure, dynamic QR code for transactions.
-   **Real-time Symmetric UI Updates:** Using Socket.IO to broadcast an event () from the server to ensure both driver and passenger UIs react simultaneously to a single action (QR scan).
-   **Backend Performance Optimization:** Using in-memory caching () to dramatically speed up repetitive database lookups.
-   **Non-Blocking UI:** Designing UI flows where background tasks (like location checks) run in parallel, providing immediate feedback to the user instead of showing a loading spinner.
